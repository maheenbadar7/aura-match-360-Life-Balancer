# aura-match-360-Life-Balancer
A Python-based wellness application designed to analyze user energy metrics, calculate lifestyle compatibility, and provide actionable scores to optimize 360-degree balance.
A creative Python-based matching engine that analyzes user interests and energy levels to determine compatibility scores. This project demonstrates foundational Python skills including dictionary management, user input handling, and algorithmic logic.

##  Features
- **Input Processing:** Captures and sanitizes user data.
- **Matching Algorithm:** A custom scoring system based on interest alignment and energy proximity.
- **Modular Code:** Built using functions for high readability and maintainability.

## Built With
- **Python 3.x**
- **Standard Libraries:** `math` (logic implementation)

## 📂 Project Structure
- `main.py`: The core execution script containing the matching logic.
- `README.md`: Documentation and project overview.

## 💡 How to Run
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/auramatch.git`
2. Run the script: `python main.py`
